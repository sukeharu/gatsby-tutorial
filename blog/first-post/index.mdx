# スコープ

スコープとは変数などオブジェクトの「アクセス可能範囲」のこと。原則として内側のスコープから外側のスコープを参照することはできるが、その逆はできない。範囲の大きいスコープから順に、JavaScirptには3つのスコープがある。例えば、関数スコープからグローバルスコープの参照はできるが、グローバルスコープから関数スコープの参照はできない。
